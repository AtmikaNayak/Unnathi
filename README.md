# Unnathi 🚀  
Built at **Her-a-thon 2026 | NMAM Institute of Technology**

Unnathi is a web platform designed to help women discover scholarships, government schemes, internships, and personal finance resources in one place.

---

## 🌟 Problem
Many scholarships, government schemes, and career opportunities for women exist, but the information is scattered across different websites.  
Students often miss these opportunities simply because they do not know where to find them.

At the same time, financial literacy among young women is still limited.

---

## 💡 Solution
**Unnathi** solves this by creating a single platform where women can:
- Discover scholarships and opportunities
- Explore government schemes for women
- Find internships
- Learn basic financial literacy
- Ask an AI assistant for quick guidance

---

## 🚀 Features
### 🎓 Scholarship Discovery
Browse and discover scholarships available for women and students.

### 🏛 Government Schemes
Explore government programs designed to support women empowerment.

### 💼 Internship Opportunities
Find internship opportunities to gain real-world professional experience.

### 💰 MoneyLab
Learn basic personal finance topics such as:
- Saving
- Budgeting
- Investing


Users can ask questions like:

```
What scholarships are available for women in engineering?
How can I start investing as a student?
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/herlaunch.git
cd herlaunch
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
